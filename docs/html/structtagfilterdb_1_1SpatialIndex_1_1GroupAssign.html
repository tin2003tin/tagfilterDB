struct tagfilterdb::SpatialIndex< DataType, Dimensions, MaxChildren, MinChildren, RangeType, AreaType >::GroupAssign

Assignment of nodes to groups during splitting.

This struct is used to assign nodes to two groups during the node splitting process. It includes an array for group assignments and two groups for managing the split.
