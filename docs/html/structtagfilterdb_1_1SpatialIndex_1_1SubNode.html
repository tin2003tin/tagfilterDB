struct tagfilterdb::SpatialIndex< DataType, Dimensions, MaxChildren, MinChildren, RangeType, AreaType >::SubNode

Represents a sub-node in the spatial index.

A sub-node contains a bounding box, an optional child node, and associated data. It represents a data entry or a child node in the index.
