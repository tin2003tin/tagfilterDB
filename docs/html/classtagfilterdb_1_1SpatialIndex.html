class tagfilterdb::SpatialIndex< DataType, Dimensions, MaxChildren, MinChildren, RangeType, AreaType >

A spatial index for querying and inserting multidimensional data.

This class represents a spatial index structure that supports efficient querying and insertion of multidimensional data. It can handle varying numbers of dimensions and provides functionality for managing nodes and sub-nodes.

Template Parameters
DataTypeThe type of data stored in the index.
DimensionsThe number of dimensions in the spatial index.
MaxChildrenMaximum number of children per node.
MinChildrenMinimum number of children per node.
RangeTypeType used for range calculations.
AreaTypeType used for area calculations.
