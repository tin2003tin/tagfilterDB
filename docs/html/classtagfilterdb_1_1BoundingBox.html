Detailed Description

template<int Dimensions, class RangeType = double, class AreaType = double>
class tagfilterdb::BoundingBox< Dimensions, RangeType, AreaType >

A template class for representing a multi-dimensional bounding box.

The BoundingBox class stores the minimum and maximum coordinates for each dimension, defining a hyper-rectangle in a multi-dimensional space. It provides utility like access, overlap, union, expand, etc

Template Parameters
DimensionsThe number of dimensions for the bounding box.
RangeTypeThe data type for the coordinates (e.g., int, float, double).
AreaTypeThe area data type for the coordinates (e.g., int, float, double).

using tagfilterdb::BoundingBox< Dimensions, RangeType, AreaType >::EDGE = std::pair<RangeType, RangeType>

Alias for a pair representing the minimum and maximum bounds in a dimension.

This alias is used to represent an edge of the bounding box in a specific dimension. The first element of the pair is the minimum value, and the second element is the maximum value.
