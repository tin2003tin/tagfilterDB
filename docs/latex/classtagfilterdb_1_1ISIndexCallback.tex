class tagfilterdb\+::\+ISIndex\+Callback$<$ SIndex $>$}
Interface for callback functions used in spatial index operations. 

This class is used to define callbacks that can process the results of spatial index operations such as searches. It is a template class where {\ttfamily SIndex} is the spatial index type.
