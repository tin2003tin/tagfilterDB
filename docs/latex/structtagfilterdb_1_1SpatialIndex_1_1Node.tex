struct tagfilterdb\+::\+Spatial\+Index$<$ Data\+Type, Dimensions, Max\+Children, Min\+Children, Range\+Type, Area\+Type $>$\+::\+Node}
Represents a node in the spatial index tree. 

A node contains an array of sub-\/nodes and information about its height and the current number of children. It provides functionality to check if the node is a leaf node.
