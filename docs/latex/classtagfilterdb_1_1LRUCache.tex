class tagfilterdb\+::\+LRUCache$<$ Key, Value $>$}
A Least Recently Used (LRU) Cache implementation. 

This class manages a cache using the LRU eviction policy. It stores key-\/value pairs and ensures that the least recently used items are evicted when the cache exceeds its total charge limit.
