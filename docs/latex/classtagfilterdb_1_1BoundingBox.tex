A template class for representing a multi-\/dimensional bounding box. 

The {\ttfamily \doxylink{classtagfilterdb_1_1BoundingBox}{Bounding\+Box}} class stores the minimum and maximum coordinates for each dimension, defining a hyper-\/rectangle in a multi-\/dimensional space. It provides utility like access, overlap, union, expand, etc
