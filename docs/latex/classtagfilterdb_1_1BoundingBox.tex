class tagfilterdb\+::\+Bounding\+Box$<$ DIMS, RANGETYPE $>$}

A template class for representing a multi-\/dimensional bounding box. 

The {\ttfamily \mbox{\hyperlink{classtagfilterdb_1_1BoundingBox}{Bounding\+Box}}} class stores the minimum and maximum coordinates for each dimension, defining a hyper-\/rectangle in a multi-\/dimensional space. It provides utility like access, overlap, union, expand, etc


\begin{DoxyTemplParams}{Template Parameters}
{\em DIMS} & The number of dimensions for the bounding box. \\
\hline
{\em RANGETYPE} & The data type for the coordinates (e.\+g., {\ttfamily int}, {\ttfamily float}, {\ttfamily double}). \\

\end{DoxyTemplParams}
